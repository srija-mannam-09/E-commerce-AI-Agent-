# E-commerce-AI-Agent-

•	Developed an AI-powered FastAPI agent using LLaMA 3 (Ollama) that converts natural language e-commerce queries into executable SQL on a local SQLite database, delivering real-time, streamed responses.


•	Engineered a modular, production-ready architecture with automated data loading, API testing via cURL + Swagger UI, and a live typing simulation for an interactive user experience.
